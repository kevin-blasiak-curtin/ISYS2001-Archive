An introductory coursework repository for learning programming fundamentals at Curtin University. This repository serves as a structured learning path for students taking their first steps into programming — covering the GitHub/Colab environment, core Python principles, and AI-assisted coding practices — building toward more advanced concepts like data manipulation with pandas.

## Description

This is an **introductory** unit — no prior programming experience is assumed. Its purpose is to get students comfortable with three things at once:

1. **The environment** — working with notebooks in GitHub and Google Colab, the tools they'll use throughout the unit and beyond
2. **Programming fundamentals** — core Python concepts (variables, logic, loops, functions, data structures) taught step by step
3. **AI-assisted coding** — using AI tools (e.g. Gemini, GitHub Copilot, ChatGPT) responsibly and effectively as part of a modern programming workflow, not as a replacement for understanding the fundamentals

The content is organised into modules that progressively build these skills, from a student's very first lines of code through to working with real data using pandas. Each module includes hands-on activities and mini-projects designed to give students the practical tools and confidence to continue their programming journey.

## Getting Started

This unit uses **Google Colab** exclusively — no local installation, virtual
environments, or package management required.

### Prerequisites

- A Google account (for Colab and Google Drive)
- A web browser

That's it — no Python, Jupyter, or local setup needed.

### Opening a Notebook in Colab

1. Navigate to the notebook file on GitHub
2. Click **"Open in Colab"** at the top of the notebook (see badge example below), or manually replace `github.com` with `colab.research.google.com/github` in the notebook's URL
3. Once open in Colab, go to **File > Save a copy in Drive** to create your own editable copy — the version opened directly from GitHub is read-only
4. Run cells sequentially from top to bottom using **Shift+Enter** or the ▶️ button

All required libraries (pandas, numpy, matplotlib, etc.) are pre-installed in the Colab environment — no `pip install` needed.


## Usage

### Getting Started with the Content

Begin with Module 01 to establish Python programming fundamentals:

1. Open `01_Hello_Python!_Your_First_Steps_in_Coding.ipynb` in Colab for basic syntax
2. Progress through `02_Fixing_Mistakes_in_Python.ipynb` to learn debugging
3. Review `03_Making_Your_Python_Code_Clear_and_Readable.ipynb` for best practices
4. Complete the mini-project to apply learned concepts

### Module Progression

Each module builds upon previous knowledge:
- **Modules 01-02**: Python basics and user interaction
- **Modules 03-04**: Control structures and loops
- **Modules 05-06**: Functions and data organization
- **Module 07**: Data analysis with pandas

### Mini-Projects

Each module includes practical mini-projects:
- **AI Budget Tracker**: Apply Python basics to financial tracking
- **Finance Tracker Profile Setup**: User input and data validation
- Additional projects in subsequent modules

## Learning Objectives

By completing this coursework, students will:
- Become comfortable working in the GitHub + Google Colab environment
- Understand core Python programming principles (variables, control flow, loops, functions, data structures)
- Learn to use AI coding assistants effectively and responsibly as part of their workflow
- Develop foundational debugging and problem-solving skills
- Get an introduction to data manipulation using pandas
- Build the practical tools and confidence to continue their programming journey beyond this unit
